# 🔥 Dr.Workout (전공의 득근 일지)

바쁜 일정 속에서도 효율적으로 근성장을 도모하기 위해 제작한 **초경량 모바일 운동 일지**입니다.
별도의 서버나 DB 없이 브라우저의 `LocalStorage`를 활용하며, 아이폰/갤럭시 모두에서 **앱처럼 설치(PWA)**하여 사용할 수 있습니다.

## 📱 Features
- **Serverless:** 서버 비용 0원. HTML 파일 하나로 동작합니다.
- **PWA Support:** 모바일 브라우저에서 '홈 화면에 추가' 시, 주소창 없는 네이티브 앱처럼 실행됩니다.
- **Data Persistence:** 데이터는 기기 내 브라우저 저장소(LocalStorage)에 안전하게 보존됩니다.
- **Customizable:** 3분할 루틴(등/가슴/하체)이 기본 설정되어 있으며, 자유롭게 수정 가능합니다.

## 🚀 How to Use
1. 이 저장소의 [Live Demo 링크](https://본인아이디.github.io/저장소이름)를 클릭하세요.
2. 모바일 브라우저(Safari/Chrome) 공유 버튼을 누릅니다.
3. **'홈 화면에 추가'**를 선택하세요.
4. 바탕화면에 생성된 아이콘을 눌러 운동을 기록합니다.

## 🛠 Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
